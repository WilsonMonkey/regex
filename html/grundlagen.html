Die Geschichte und grundlegenden Informationen über Reguläre Ausdrücke.

Allgemeines

Was sind Regex?

Regex Kurzform für Reguläre Ausdrücke. Mit diese ist es möglich eine Zeichenkette schnell anhand von Suchkritierien (Regex) zu durchsuchen/filtern.
Textverarbeitungsprogramme, Betriebssysteme, und die meisten gängigen Programmiersprachen haben Regulare Ausdrücke Eigenschaften implementiert.

Geschichte

Stephen Cole Kleene

Begriff geht auf den Mathematiker Stephen Cole Kleene zurück (reguläre Mengen). Kleene gilt als Begründer der theoretische Informatik

Verwendung

Nach fester syntaktischer Reihenfolge von Charakteren ist ein Regex zum Durchsuchen/Filtern einer Zeichenkette.
Pro Charakter der Suchstrings wird versucht das Filterkriterium anzulegen. Beim ersten match des Suchstrings auf den kompletten Regex, wird dieser zurückzugeben und die Suche wird beendet. Selbst wenn im späteren Text noch ein besserer bsp. speziferischen Treffer gibt.

Regex Vorgang

Zuerst wird dem Regextool der Reguläre Ausdruck angegeben. Als Zweites wird der Target String übergeben. Im Regulären Ausdruck kennzeichnet der Delimiter den Anfang und das Ende. Dazwischen sind entweder literalen sprich wörtlichen Charakter, Metacharakter und Operatoren mit denen der Target String gefiltert werden soll. Wichtig ist dabei zu beachten das im Standard der erste Suchtreffer zurückgeben wird egal ob nachfolgende vielleicht noch besser auf der Suchkriterium passen würden.
